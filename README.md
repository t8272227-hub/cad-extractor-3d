# 🗺 CAD Extractor 3D

Профессиональное геодезическое веб-приложение для работы с DXF-файлами.

![Node.js](https://img.shields.io/badge/Node.js-18+-green)
![Express](https://img.shields.io/badge/Express-4.18-blue)
![License](https://img.shields.io/badge/License-MIT-yellow)

---

## ✨ Возможности

### 📐 Работа с DXF
- Загрузка и отображение DXF-чертежей
- Управление слоями (видимость, цвет)
- Вложенный DXF с геопривязкой (2D Helmert-трансформация)
- Привязка к узлам, линиям и серединам отрезков

### 📍 Геодезия
- Отметка точек с X/Y/Z координатами
- Режимы вешки: Абс. / Отн. / Призма / От поверхности
- Расчёт Z через высоту вешки, поправку, доп. высоту
- Интерполяция Z между точками
- Горизонтали с настройкой цвета и прозрачности

### 📏 Измерения
- Размерные линии
- Контур площади (свободные точки + привязка)
- Вычет колодцев и столбов из площади
- Добавление объёма свай к бетону
- Закрашивание материалами: бетон, песок, щебень, глина, грунт, асфальт, кирпич, металл

### 📋 Условные обозначения
- Стена (по центру / наружной / внутренней)
- Свая, Столб/Колонна (круг/квадрат), Колодец
- Канализация, Водопровод, Газопровод, Теплосеть, Кабель, Забор
- Выноски с параметрами для всех знаков
- Ведомость объектов (редактируемая таблица)

### 🧊 3D вид
- Триангуляция Делоне по Z-отметкам
- DXF-линии, точки-сферы, сваи-цилиндры, стены-боксы
- Закрашенные контуры в 3D

### 📄 PDF-отчёт
- Настраиваемый штамп ГОСТ: организация, объект, разработал, геодезист, проверил
- Таблица координат (2 колонки при > 12 точках)
- Таблица размеров, объёмов, материалов
- Ведомость объектов
- Рамка выбора области для кадрирования

## 🚀 Быстрый старт

### Локально

```bash
# 1. Клонировать репозиторий
git clone https://github.com/YOUR_USERNAME/cad-extractor-3d.git
cd cad-extractor-3d

# 2. Установить зависимости
npm install

# 3. Запустить
npm start
# → http://localhost:3000
```

### Разработка (авто-перезапуск)

```bash
npm run dev
```

## ☁️ Деплой на Render

### Вариант 1 — через render.yaml (рекомендуется)

1. Форкнуть/загрузить репозиторий на GitHub
2. Перейти на [render.com](https://render.com) → **New** → **Web Service**
3. Подключить GitHub-репозиторий
4. Render автоматически обнаружит `render.yaml`
5. Нажать **Deploy** — сайт будет доступен через 2-3 минуты

### Вариант 2 — ручные настройки Render

| Параметр | Значение |
|---|---|
| Environment | **Node** |
| Build Command | `npm install` |
| Start Command | `npm start` |
| Plan | Free |

> **Важно:** Render бесплатный план "засыпает" после 15 минут неактивности. Первый запрос после сна занимает ~30 секунд.
